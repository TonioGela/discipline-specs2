discipline-specs2 - Specs2 Integration for Discipline

Quick Start

To use discipline-specs2 in an existing SBT project with Scala 2.11 or a later version, add the following dependencies to your build.sbt depending on your needs:

libraryDependencies ++= Seq(
  "org.typelevel" %% "discipline-specs2" % "<version>"
)
